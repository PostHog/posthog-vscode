**1. Add isDarkTheme() helper** near the top of the script (after the `esc()` function, around line 20):

```js
function isDarkTheme() {
    return document.body.classList.contains('vscode-dark') || document.body.classList.contains('vscode-high-contrast');
}
```

VS Code webviews automatically add `vscode-dark`, `vscode-light`, or `vscode-high-contrast` to the body element.
